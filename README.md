# App Deployment Project

This repository contains documentation and scripts created during a cloud and DevOps training course. The project demonstrates the process of deploying and managing a web application using AWS infrastructure and common DevOps tools.

The goal of this project was to learn how to deploy applications in the cloud, automate server setup, configure web servers and databases, and use AWS features such as custom images and auto scaling.

---

# Project Overview

The project involves deploying a **Node.js application** on an **AWS EC2 instance**, connecting it to a **MongoDB database hosted on a separate EC2 instance**, and configuring **Nginx as a reverse proxy**.  

Infrastructure setup was automated using **bash scripts**, and AWS services such as **Launch Templates**, **Auto Scaling Groups**, and **CloudWatch monitoring** were explored.

---

# Topics Covered

This project covers the following concepts:

- Cloud computing fundamentals
- SSH and key pair authentication
- Virtual machines and EC2 instances
- Security groups and network ports
- Deploying a Node.js application on EC2
- Installing and configuring Nginx
- Reverse proxy implementation
- MongoDB database setup on a separate instance
- Environment variables for application configuration
- Running applications in the background using PM2
- Creating custom AMI images
- AWS monitoring with CloudWatch
- Launch Templates
- Auto Scaling Groups and horizontal scaling
